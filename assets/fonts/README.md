# Fonts

This project renders plates with **Sarun's ThangLuang** — the canonical Thai
highway-signage typeface, also used on real Thai government license plates.

## Download

The font is free for commercial use but its license **does not permit
redistribution of the font file**, so it is not checked into this repo.

Download it from the author's page:

- <https://www.f0nt.com/release/saruns-thangluang/>

Extract the ZIP and place the TTF here:

```
assets/fonts/SarunsThangLuang.ttf
```
